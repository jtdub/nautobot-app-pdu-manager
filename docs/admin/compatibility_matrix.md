# Compatibility Matrix

The minimum supported Nautobot version is driven by PDU Manager's use of the Nautobot 3.1
UI job-modal button API for the device-page PDU Power dropdown. The app supports the 3.x
release line (it does not yet support Nautobot 4.0).

| Pdu Manager Version | Nautobot First Support Version | Nautobot Last Support Version |
| ------------- | -------------------- | ------------- |
| 1.0.X         | 3.1.0                | 3.99.99        |
